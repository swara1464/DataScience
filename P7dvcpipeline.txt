STEP 0: CHECK AND CLEAN EXISTING PYTHON / DVC
Check Python:
python3 --version

Check pip:
pip3 --version

Check DVC:
dvc --version

If DVC is broken or missing, reinstall.
Remove DVC (only if required):
pip3 uninstall dvc -y


STEP 1: INSTALL REQUIRED TOOLS
Update system:
sudo apt update

Install Python tools (if not present):
sudo apt install python3 python3-pip -y

Install DVC:
pip3 install dvc

Verify:
dvc --version

Expected output:
DVC version x.x.x


STEP 2: CREATE PROJECT DIRECTORY
mkdir dvc-project
cd dvc-project

Initialize Git:
git init
git branch -M main

Initialize DVC:
dvc init

This creates:
.dvc/
.dvcignore


STEP 3: CREATE RAW DATA (DATA INGESTION)
Create data directory:
mkdir data

Create raw dataset:
nano data/raw_data.csv


Paste:
id,value
1,10
2,20
3,
4,40
5,50

Save and exit.
Track raw data with DVC:
dvc add data/raw_data.csv

Add metadata to Git:
git add data/raw_data.csv.dvc .gitignore .dvc
git commit -m "Add raw dataset using DVC"


STEP 4: CREATE DATA CLEANING SCRIPT
Create script directory:
mkdir src

Create cleaning script:
nano src/clean_data.py

Paste exactly:
import pandas as pd

df = pd.read_csv("data/raw_data.csv")
df = df.dropna()
df.to_csv("data/processed_data.csv", index=False)

Save and exit.


STEP 5: CREATE DVC PIPELINE (dvc.yaml)
Create pipeline file:
nano dvc.yaml

Paste exactly (indentation matters):
stages:
  clean:
    cmd: python src/clean_data.py
    deps:
      - src/clean_data.py
      - data/raw_data.csv
    outs:
      - data/processed_data.csv

Save and exit.


STEP 6: RUN PIPELINE USING DVC
Execute pipeline:
dvc repro

Expected output:
Running stage 'clean'
Generating 'data/processed_data.csv'

Verify processed data:
cat data/processed_data.csv

Expected output:
id,value
1,10
2,20
4,40
5,50

STEP 7: TRACK PROCESSED DATA WITH DVC
git add dvc.yaml dvc.lock
git commit -m "Add DVC pipeline for data cleaning"


STEP 8: VERIFY REPRODUCIBILITY
Delete processed data manually:
rm data/processed_data.csv

Reproduce pipeline again:
dvc repro

Verify file is regenerated:
ls data

Expected:
raw_data.csv
processed_data.csv

This proves reproducibility.

STEP 9: SHOW IMPORTANT COMMANDS TO EXAMINER
dvc add data/raw_data.csv
dvc repro
git status
